# 🩸 Blood Bank Management System

A console-based Blood Bank Management System developed in C++ as a First Semester project.

## Features
- Admin Login Authentication
- Add Donor Records
- Search Donors
- Search by Blood Group
- Search by Location
- Update Donor Information
- Delete Donor Records
- File Handling using Text Files
- Password Encryption

## Technologies Used
- C++
- File Handling
- Object-Oriented Programming (OOP)

## Files
- bbms.cpp - Source code
- bbms.exe - Executable application
- credentials.txt - Admin credentials
- donors.txt - Donor database
- final report.pdf - Project report

## How to Run
1. Download all project files.
2. Keep `bbms.exe`, `credentials.txt`, and `donors.txt` in the same folder.
3. Double-click `bbms.exe` to start the program.
